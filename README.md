# eotobi.dev

Personal portfolio built with React.

## Quick Start

```bash
npm install
npm start
```

## Customize

Edit `src/portfolio.js` for content and `src/theme.js` for colors.

## Deploy

```bash
npm run deploy
```

Pushes a production build to the `gh-pages` branch. Enable GitHub Pages in your repo settings and point it to that branch.

## Stack

React 18 · React Router 6 · Styled Components · BaseUI
